# 公司品牌视觉指南

## 品牌口号

**危机 · 专业 · 专注 · 创新**

## 品牌色

| 用途 | 色值 | 色块 |
|------|------|------|
| 主色（深蓝） | `#003D7A` | 封面标题背景、重点色块 |
| 辅色（科技蓝） | `#0066CC` | 图表主色、强调文字 |
| 辅色（浅蓝） | `#E8F2FA` | 封面副标题底色 |
| 文字色 | `#333333` | 正文 |
| 标题色 | `#1A1A1A` | 各级标题 |
| 装饰色（青） | `#00A8A8` | 图表辅色、数据高亮 |
| 装饰色（橙） | `#F5A623` | 警示/重点关注标记 |

## 字体

| 用途 | 字体 | 备选 |
|------|------|------|
| 封面主标题 | 微软雅黑 Bold | 思源黑体 Bold |
| 章节标题 | 微软雅黑 Bold | 思源黑体 Bold |
| 正文 | 微软雅黑 Regular | 思源黑体 Regular |
| 封面副标题/日期 | 微软雅黑 Light | 思源黑体 Light |
| 页码 | 微软雅黑 Regular, 10pt |

## 字号规范

| 元素 | 字号 |
|------|------|
| 封面主标题 | 36pt |
| 封面副标题 | 16pt |
| 章节标题 | 28pt |
| 内容标题 | 22pt |
| 正文 | 16pt |
| 注释/来源 | 10pt |
| 页码 | 10pt |

## Logo

- 封面右下角放置公司 Logo（`logo.png`）
- Logo 尺寸：约 1.3cm × 0.4cm
- 位置：右下角，距边缘约 0.5cm
