# 常用 PPT 结构模板

## 1. 周/月工作汇报

```
封面（1页）
目录（1页）         ← 可选，内容多时加
├── 上周/月工作总结（1-2页）
│   ├── 已完成事项
│   ├── 关键数据/指标
│   └── 亮点成果
├── 本周/月工作计划（1-2页）
│   ├── 重点任务
│   ├── 时间节点
│   └── 需要的支持
├── 问题与风险（1页）
│   ├── 遇到的问题
│   └── 解决方案
└── 结束页（1页）
```

## 2. 技术分享

```
封面（1页）
├── 背景/问题（1-2页）    ← 为什么需要这个技术
├── 核心概念（2-3页）      ← 是什么
├── 实践/方案（2-4页）     ← 怎么做
│   ├── 架构设计
│   ├── 关键实现
│   └── 踩坑记录
├── 效果/数据（1-2页）     ← 效果如何
├── 总结与展望（1页）
└── 结束页 + Q&A（1页）
```

## 3. 项目复盘

```
封面（1页）
├── 项目概况（1页）
│   ├── 目标
│   ├── 团队
│   └── 时间线
├── 成果展示（2-3页）
│   ├── 核心数据
│   └── 对比目标
├── 过程回顾（2-3页）
│   ├── 关键节点
│   └── 决策记录
├── 经验教训（1-2页）
│   ├── 做得好的
│   └── 待改进的
├── 下一步行动（1页）
└── 结束页（1页）
```

## 4. 季度/年度总结

```
封面（1页）
├── 核心数据概览（1页）
├── 各业务线/项目成果（3-5页）
├── 关键里程碑（1-2页）
├── 团队建设（1页）
├── 问题与反思（1页）
├── 下季度/年度规划（2-3页）
└── 结束页（1页）
```

## 5. 产品介绍/宣讲

```
封面（1页）
├── 开场 Hook（1页）       ← 数据/场景/问题
├── 产品定位（1页）         ← 一句话 + 目标用户
├── 核心功能（2-3页）       ← 每个功能 1 页
├── 技术架构（1-2页）       ← 技术团队内部看
├── 竞品对比（1页）         ← 选配
├── 案例/客户（1-2页）
├── 下一步计划（1页）
└── 结束页 + 联系方式（1页）
```

## 通用页面布局映射

| 内容类型 | 推荐布局 |
|----------|----------|
| 封面 / 结束页 | 封面样式（深蓝底 + 白色大字） |
| 章节过渡页 | `节标题` 布局 |
| 正文（单一主题） | `标题和内容` 布局 |
| 对比（前后/优劣/AB） | `比较` 布局 |
| 两列展示（图文/并列） | `两栏内容` 布局 |
| 图表 / 大图 | `空白` 布局（手动放置） |
